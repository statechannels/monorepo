## Rock Paper Scissors: A Force-Move Games POC

### Developement Info

#### To run a dev server:

`npm run start`

#### To build:

`npm run build`

#### To run tests:

`npm test`

#### Project style

Please use the sublime package _JsPrettier_ for formatting. All of the defaults should make sense except for the following:

```
  "printWidth": 100,
  "singleQuote": true,
  "trailingComma": "all",
```
