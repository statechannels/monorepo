## Rock Paper Scissors: A Force-Move Games POC

### Developement Info

#### To run a dev server:

`npm run start`

#### To build:

`npm run build`

#### Project style

Please use the sublime package *JsPrettier* for formatting. All of the defaults should make sense except for the following:
```
  "printWidth": 100,
  "singleQuote": true,
  "trailingComma": "all",
```
